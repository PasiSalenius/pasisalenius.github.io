AreenaX for Mac

AreenaX is a little Yle Areena player for the Mac. Itâ€™s made possible by the Yle API, that is freely available to all developers.

All media playback is done natively by OS X with hardware acceleration so Adobe Flash is not required. Programs can be beamed to an Apple TV over AirPlay.

Just for fun included are some video filtering options like contrast, brightness and saturation, with a gamma correction slider.

AreenaX for Apple TV

AreenaX is also available on the new Apple TV, in addition to the Mac version. It lets you enjoy the content on the big screen with a fast and responsive tvOS interface controlled using the touch remote.

AreenaX is a native tvOS app so it can run full-speed on the Apple TV hardware and use completely custom animations, layout and styling. It has also been written entirely in Swift.
